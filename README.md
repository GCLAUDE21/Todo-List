# TODO LIST

Projet interactif réalisé dans le cadre de ma formation Développeur Web.

**Demo live**
[Voir la page](https://gclaude21.github.io/Todo-List/)

## Objectif

Créer une todo list en JavaScript natif : l'utilisateur peut ajouter des tâches via un formulaire, les supprimer au clic, et retrouver ses tâches après rechargement de la page grâce au localStorage.

## Compétences travaillées

- Manipulation du DOM avec JavaScript natif
- Gestion des événements (submit, click)
- Persistance des données avec localStorage (setItem, getItem)
- Logique conditionnelle pour l'affichage par défaut

## Stack technique

JavaScript natif. HTML natif. CSS natif. Aucun framework ou bibliothèque externe.
